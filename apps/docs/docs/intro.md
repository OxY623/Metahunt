# MetaHunt · Prolog

_Говорит ROOT (ADMIN). Старый «system-protocol» закрыт. С этого пролога мы перезагружаем город и правила — без лишней шелухи._

MetaHunt — городская MMO про влияние, события и живые встречи. Ты входишь в город как гость, а роль выбираешь сам, когда готов. Выбор фиксирует интерфейс, способности и то, как город реагирует на тебя.

## MVP статус (актуально)

- Единый `SiteHeader` с выезжающим меню
- Общий `SiteFooter` на всех страницах, включая чат
- Страница настроек пользователя и системных параметров UI
- Профиль: `avatar`, `bio`, `privacy`
- Компактный режим и уменьшение анимаций
- Чат с header + sticky footer
- Центрирование layout через `page-shell` на больших экранах

## Кодекс ROOT

1. **Prolog вместо протокола.** Есть один голос — ADMIN/ROOT — он устанавливает стартовые правила.
2. **Выбор после регистрации.** Регистрация открывает доступ. Архетип (Wolf/OXY, FoxY, Bear, Owl) выбираешь осознанно позже — это точка невозврата.
3. **Короткие правила.** Чaсть за организацию событий. Налог за ресурсы. Анонимность стоит Shards. Нарушил — бан без голосования.

## Что за мир

- Город делится на четыре архетипа: Wolf/OXY (стратегия), FoxY (социальный взлом), Bear (ресурсный контроль), Owl (информация и компромат).
- Экономика строится на внутренней валюте Shards: скиллы, блокировки, доступы, штрафы.
- Игровые петли: ищи события, подключай людей, усиливай влияние, защищай трафик или продавай данные.

## Быстрый старт

```bash
# Установка зависимостей
pnpm install

# Запуск API (FastAPI)
cd apps/api && pnpm dev

# Запуск фронтенда (Next.js)
cd apps/web && pnpm dev

# Запуск документации
cd apps/docs && pnpm start
```

Или из корня:

```bash
pnpm dev
```

### Если pnpm недоступен в WSL

```bash
cd apps/api
./venv/bin/python -m uvicorn main:app --reload --port 8000
```

## Переменные окружения

### apps/web

- `NEXT_PUBLIC_API_URL` — URL бэкенда (по умолчанию `http://localhost:8000`)

### apps/api

- `DATABASE_URL` — PostgreSQL connection string
- `SECRET_KEY` — JWT секрет
- `ALGORITHM` — HS256
