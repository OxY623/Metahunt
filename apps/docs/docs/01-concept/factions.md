# Фракции

На старте пользователь выбирает одну из 4 фракций. Выбор необратим (без штрафа).
Каждая фракция — это архетип, стиль игры и уникальные скиллы.

---

## Матрица противостояний

```
FOXY  контрит  OXY   (хитрость / глитч экрана)
BEAR  контрит  FOXY  (ресурсы / «защита»)
OXY   контрит  BEAR  (моральное превосходство / суды)
OWL   контрит  всех  (продаёт компромат на любого)
```

## Первый квест: сюжетная развилка

После регистрации Волку предлагается выбор:

**Вариант A — Посылка из прошлого**

Отправить Лисе зашифрованное сообщение. Она выбирает: расшифровать (риск — Медведь увидит) или удалить (теряет XP).

**Вариант B — Взлом корпората**

Через Сову написать от имени Лисы в официальный канал Медведя — он начнёт ей не доверять.

**Вариант C — Тихое наблюдение**

10 минут в чате Лисы незамеченным — получить данные о её уровне защиты.

:::tip
Выбор квеста влияет на стартовые бонусы и первые отношения с Совой.
:::
