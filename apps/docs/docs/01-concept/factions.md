# Фракции

На старте пользователь выбирает одну из 4 фракций. Выбор необратим (без штрафа).
Каждая фракция — это архетип, стиль игры и уникальные скиллы.

---

## Логика отношений (дискретная математика)

Обозначим множество архетипов:

```
A = { FOXY, OXY, BEAR, OWL }
```

Задаём отношения между архетипами как множества пар (R ⊆ A×A):

```
R_counter = { (FOXY,OXY), (BEAR,FOXY), (OXY,BEAR) }
R_trade   = { (OWL,x) | x ∈ A }
I_ally    = { (a,a) | a ∈ A }
```

Свойства:
- `R_counter` — антисимметрично и иррефлексивно (если a контрит b, то b не контрит a).
- `I_ally` — рефлексивно (каждый союзник сам себе).
- `R_trade` направлено от OWL ко всем, это не «контр», а экономическая связь.

Это значит, что скиллы работают **по направлению ребра**: контр‑фракция может применить ударные фишки, а не наоборот.

---

## Матрица связей

```
SOURCE → TARGET | FOXY | OXY  | BEAR | OWL
------------------------------------------
FOXY            | ally | counter | neutral | neutral
OXY             | neutral | ally | counter | neutral
BEAR            | counter | neutral | ally | neutral
OWL             | trade | trade | trade | ally
```

---

## Чат и клики

- Цель выбирается **кликом по сообщению** в чате.
- Без ввода UID: навыки активируются по выбранной цели.
- Навыки с направлением контра включаются **только** если связь = `counter`.
- `Золотой щит` даёт временный иммунитет к контр‑навыкам (5 минут).
- `Шёпот` — анонимный DM в текущей комнате: виден только отправителю и цели.

---

## Интерактивные умения (чат-эффекты, vNext)

Все умения ниже срабатывают через чат и подтверждаются системным событием. Состояния эффектов хранятся как персистентные записи с TTL, чтобы UI мог показать таймер и активные статусы.

- **Wolf/OXY:** `Маркер охоты` — помечает цель на 3 минуты, её сообщения подсвечиваются и доступны всем твоим союзникам. `Разрез канала` — на 60 секунд снижает вес реплик цели (её сообщения уходят в конец ленты).
- **FoxY:** `Вуаль` — отправитель становится невидимым в списке участников комнаты на 2 минуты. `Подмена реплики` — первое сообщение цели после активации получает нейтральный префикс-стикер (скрывает эмоцию/реакцию).
- **Bear:** `Печать доступа` — блокирует использование активных умений целью на 90 секунд. `Инкассация` — разовый налог при входе в комнату (видно как системное событие).
- **Owl:** `Тихий лог` — 5 минут собирает метаданные о реакции на сообщения цели (кол-во ответов/реакций). `Слепая сделка` — отправляет анонимный запрос на обмен данными в личку.

Подробные правила хранения и доставки событий: см. `apps/docs/docs/02-architecture/chat-events.md`.

## UI-подсказки для архетипов

- **OXY:** холодный неон, акцент на кнопки давления, статус контра подсвечен ярче.
- **FOXY:** вспышки и глитч, короткие уведомления, анонимные метки.
- **BEAR:** тяжёлые панели, золото и графит, таймеры блокировок крупнее.
- **OWL:** тёмные фоны, мягкие подсветки, скрытые метки и нейтральные статусы.

## Первый квест: сюжетная развилка

После регистрации Волку предлагается выбор:

**Вариант A — Посылка из прошлого**

Отправить Лисе зашифрованное сообщение. Она выбирает: расшифровать (риск — Медведь увидит) или удалить (теряет XP).

**Вариант B — Взлом корпората**

Через Сову написать от имени Лисы в официальный канал Медведя — он начнёт ей не доверять.

**Вариант C — Тихое наблюдение**

10 минут в чате Лисы незамеченным — получить данные о её уровне защиты.

:::tip
Выбор квеста влияет на стартовые бонусы и первые отношения с Совой.
:::
