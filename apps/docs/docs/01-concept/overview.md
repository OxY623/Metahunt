# MetaHunt — Обзор проекта

> **«Наконец-то место для твоих обид»**

**MetaHunt** — это анти-социальная сеть в жанре пиксельный киберпанк. Платформа, где пользователь выбирает одну из 4 фракций и участвует в корпоративных интригах, шпионаже и экономических войнах.

---

## Жанр и атмосфера

| Параметр  | Значение                                               |
| --------- | ------------------------------------------------------ |
| Жанр      | RPG-мессенджер / социальная стратегия                  |
| Визуал    | Pixel-art + Cyberpunk (неон, дождь, тёмные переулки)   |
| Атмосфера | «Дорогой инди-хоррор» — мрачно, стильно, провокационно |
| Аудитория | 18–35, геймеры, любители интриг и киберпанка           |

---

## Эстетика

- **Шрифты:** `Press Start 2P`, `DotGothic16` (Google Fonts, бесплатно)
- **Цветовая база:** тёмный фон `#0D1117`, акценты по фракциям
- **CSS-хак:** `image-rendering: pixelated` в Tailwind — любая картинка становится «олдскульной»
- **Тени:** `4px 4px 0px 0px rgba(0,0,0,1)` — жёсткий пиксельный стиль

---

## Ключевая механика

Пользователь выбирает фракцию **один раз** при регистрации. Фракция определяет:

- цветовую тему интерфейса
- доступные скиллы
- налоговую ставку на транзакции
- кого можно атаковать, а кто атакует тебя

:::info Сменить фракцию можно, но...
Это статус **«Предатель»** — временный дебафф, зато открывает уникальные квесты.
:::
