# User Stories

Формат: **Как [роль], я хочу [действие], чтобы [цель].**

---

## Регистрация и онбординг

| #     | Story                                                                                                     | Приоритет |
| ----- | --------------------------------------------------------------------------------------------------------- | --------- |
| US-01 | Как новый пользователь, я хочу увидеть 4 карточки фракций при регистрации, чтобы понять разницу до выбора | 🔴 MVP    |
| US-02 | Как новый пользователь, я хочу получить 100 стартовых Осколков, чтобы сразу попробовать скиллы            | 🔴 MVP    |
| US-03 | Как новый пользователь, я хочу пройти первый квест, чтобы понять механику игры                            | 🟡 Фаза 2 |
| US-04 | Как пользователь, я хочу сменить фракцию с дебаффом «Предатель», чтобы попробовать другой стиль           | 🟢 Фаза 3 |
