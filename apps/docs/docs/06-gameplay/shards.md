# Экономика Осколков (Shards)

**Осколки** — единственная валюта в Metahunt.  
Всё стоит Осколков. Скиллы, баны, разбаны, анонимность.

---

## Стартовый баланс

| Параметр           | Значение       |
| ------------------ | -------------- |
| Стартовые Осколки  | 100            |
| Стартовая Энергия  | 100 / день     |
| Пополнение энергии | Каждые 24 часа |

---

## Как заработать Осколки

| Способ                            | Количество |
| --------------------------------- | ---------- |
| Ежедневный вход                   | +10        |
| Выполнение квеста                 | +25–50     |
| Кто-то купил у тебя инфу (Сова)   | +переменно |
| Лиса получает 30% налогов Медведя | +авто      |
| Купить за реальные деньги         | —          |

---

## Как тратятся Осколки

| Действие                         | Стоимость            |
| -------------------------------- | -------------------- |
| Прямой удар (OXY)                | 5                    |
| Визит Волка к Лисе (авто-налог)  | 10 (снимает Медведь) |
| Глитч экрана (FOXY)              | 15                   |
| Золотой Щит (BEAR)               | 20                   |
| Шёпот — анонимный DM (OWL)       | 20                   |
| Прослушка профиля (OWL)          | 25                   |
| Блокировка Порта / бан (BEAR)    | 30                   |
| Разбан (покупка у Медведя)       | 50+                  |
| Кофе Медведя (пополнить энергию) | 30                   |

---

## Распределение налога Медведя

```
Волк платит 10 Осколков при визите к Лисе
    ├── 7 Осколков → Казна Медведя (70%)
    └── 3 Осколка  → Лиса (30%)
```

---

## Энергия — защита от спама

```python
# Каждое действие тратит энергию
ENERGY_COSTS = {
    "message":  1,
    "skill":    5,
    "interact": 3,
    "invite":   2,
}

# Если энергия = 0 → действие заблокировано
# Восстановление: +100 каждые 24 часа
# Или купи «Кофе Медведя» за 30 Осколков
```

:::tip Монетизация через энергию
Когда у пользователя кончается энергия — показываем предложение купить «Кофе Медведя».  
Это не пэйвол, это раздражающий, но честный paywall.
:::
