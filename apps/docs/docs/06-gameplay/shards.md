# Экономика Осколков (Shards)

**Осколки** — единственная валюта в MetaHunt.  
Всё стоит Осколков: скиллы, баны, анонимность, доступы и сделки.

---

## Базовые принципы

- Осколки создают **выбор**, а не гринд.
- Любая сильная фишка имеет цену и откат.
- Экономика должна **гореть**: входы → расходы → дефицит → решение.

## Налоги и распределение

```
Волк платит 10 Осколков при визите к Лисе
    ├── 7 Осколков → Казна Медведя (70%)
    └── 3 Осколка  → Лиса (30%)
```

## Энергия — защита от спама

```python
# Каждое действие тратит энергию
ENERGY_COSTS = {
    "message":  1,
    "skill":    5,
    "interact": 3,
    "invite":   2,
}

# Если энергия = 0 → действие заблокировано
# Восстановление: +100 каждые 24 часа
# Или купи «Кофе Медведя» за 30 Осколков
```

:::tip Монетизация через энергию
Когда у пользователя кончается энергия — показываем предложение купить «Кофе Медведя».  
Это не пэйвол, это раздражающий, но честный paywall.
:::

## Анти‑абуз правила (vNext)

- **Daily cap**: максимум +120 Осколков в сутки без покупок.
- **Soft cap**: после 120 награды режутся в 2 раза.
- **Penalty**: за спам‑действия вводится коэффициент `0.6`.
- **Cooldown**: одинаковые действия в цикле теряют эффективность.

---

## Баланс и дефицит

Экономика должна держать игрока в диапазоне **10–200 Осколков**:  
меньше — скука, больше — обесценивание рисков.

**Стабилизаторы**:
- Золотой Щит и Баны — сильные, но дорогие.
- Анонимность — платная, а не бесплатная привычка.
- Разбан всегда выше стоимости бана.

## Что показывает UI

- Самые дорогие действия на экране (контроль риска).
- Предупреждение, если баланс < 15.
