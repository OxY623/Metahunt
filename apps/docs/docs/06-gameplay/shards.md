# Экономика Осколков (Shards)

:::note Статус
✅ Спецификация v0.3 (MVP, боевой баланс). Реализация в коде — далее.
:::

**Осколки** — единственная валюта в MetaHunt.  
Всё стоит Осколков: скиллы, баны, анонимность, доступы, усиления и сделки.

---

## Базовые принципы

- Осколки создают **выбор**, а не гринд.
- Любая сильная фишка имеет цену и откат.
- Экономика должна **гореть**: входы → расходы → дефицит → решение.

---

## Режимы баланса

Мы работаем в двух режимах лимитов. По умолчанию в конфликтных окнах включается **строгий (боевой)**.

| Режим | Где используется | Множитель наград | Множитель цен | Дневной кап | После капа |
| ----- | ---------------- | ---------------- | ------------- | ---------- | ---------- |
| Мягкий | онбординг, низкая активность | 1.0 | 1.0 | +120 | награды ×0.5 |
| Строгий (боевой) | сезоны, войны, ивенты | 0.7 | 1.3 | +80 | награды ×0.4 |

## Налоги и распределение

- Все налоги идут в казну Медведя.
- Лиса автоматически получает 30% от собранных налогов.
- Правило округления: дробные значения округляются вниз, остаток остаётся у Медведя.

Пример визита (мягкий / строгий):

```
ОКСИ → ФОКСИ (визит)
10 / 13 Осколков
7 / 9 → Казна Медведя
3 / 4 → Лиса (30% от налога)
```

Пример инвайта (мягкий / строгий):

```
Инвайт использован
Награда 20 / 14 Осколков
Налог 20% = 4 / 2
16 / 12 → Инвайтер
4 / 2 → Казна Медведя
1 / 0 → Лиса (30% от налога, округление вниз)
```

## Транзакции и аудит

Любое изменение Осколков — это запись в `shards_ledger`:

- `delta` со знаком `+` или `-`
- `reason` (skill, invite, tax, refund, purchase)
- `meta` (id цели, архетип, код инвайта)
- `balance_after` для UI и анти‑абуза

## Энергия — защита от спама

```python
# Каждое действие тратит энергию
ENERGY_COSTS = {
    "message":  1,
    "skill":    5,
    "interact": 3,
    "invite":   2,
    "post":     2,
    "map_ping": 3,
}

# Если энергия = 0 → действие заблокировано
# Восстановление: +100 каждые 24 часа
# Или купи «Кофе Медведя» за 30/39 Осколков
```

:::tip Монетизация через энергию
Когда у пользователя кончается энергия — показываем предложение купить «Кофе Медведя».  
Это не пэйвол, это раздражающий, но честный paywall.
:::

## Анти‑абуз правила

| Параметр | Мягкий | Строгий |
| -------- | ------ | ------- |

| Soft cap | после 120 | после 80 |
| Penalty за спам | ×0.6 | ×0.5 |
| Cooldown повторов | стандарт | усилен ×1.5 |

## Баланс и дефицит

Экономика должна держать игрока в диапазоне **10–200 Осколков**:  
меньше — скука, больше — обесценивание рисков.

Стабилизаторы:
- Золотой Щит и Баны — сильные, но дорогие.
- Анонимность — платная, а не бесплатная привычка.
- Разбан всегда выше стоимости бана.

---

## Что показывает UI

- Текущий баланс и активный режим лимитов (мягкий/строгий).
- Дневной лимит и сколько осталось до cap.
- Самые дорогие действия на экране (контроль риска).
- Предупреждение, если баланс < 15.
