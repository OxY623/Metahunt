## Маркетинговая стратегия

### 1. Закрытый клуб (Дефицит)

Вход только по инвайт-осколкам.  
Волк — 3/2 инвайта (мягкий/строгий), Лиса — 5/4.  
**Эффект:** люди клянчат инвайты → органический buzz.

### 2. Вирусный контент (TikTok / Reels)

Пиксельные ролики с диалогами Волка и Лисы.  
Заголовок: _«Он писал код, она украла ключи. На чьей стороне ты?»_

### 3. Лендинг-заглушка

Чёрный экран + бегущие строки кода.  
Пользователь оставляет почту чтобы «взломать доступ».

### 4. Скандальный слух

«B.L.Y.A.T. позволяет читать чужие переписки» —  
(это скилл Совы, но звучит как утечка данных → медиа-волна)
